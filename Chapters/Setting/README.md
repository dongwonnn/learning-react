# React Setting 모음

## 프로젝트 생성

    ```
    yarn create react-app "프로젝트 이름"
    npm install react-app "프로젝트 이름"
    ```

## VSCode 확장 프로그램

    1.  ESLint : 자바스크립트 문법, 코드 스타일 검사.
    2.  Reactjs Code Snippers : 단축 단어 자동 생성.
    3.  Prettier-Code formatter : 코드 스타일 자동 정리

## Prettier 설정

    - .prettierrc 파일
        ```
        {
        "singleQuote": true,
        "semi": true,
        "useTabs": false,
        "tabWidth": 2,
        "trailingComma": "all",
        "printWidth": 80
        }
        ```
